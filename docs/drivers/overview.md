# Drivers Overview

Task Tracker supports multiple drivers for messaging and tasks. Drivers let you swap integrations without changing your business logic.

## Driver Types
- **Messaging drivers**: parse platform payloads into `IncomingMessage`
- **Task drivers**: connect to your task manager and implement `TaskManager`
- **AI drivers**: provide an LLM client implementation
