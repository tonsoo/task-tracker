# Visão Geral

O Task Tracker recebe mensagens do WhatsApp e as transforma em ações no gerenciador de tarefas configurado usando IA. É distribuído como um pacote Laravel e integra via Service Provider com bindings, rotas e um arquivo de configuração publicável.

- Namespace do pacote: `Tonso\TaskTracker\`
- Provider de entrada: `src/TaskTrackerServiceProvider.php`
- Rotas: `routes/api.php`
- Config: `config/task-tracker.php`
